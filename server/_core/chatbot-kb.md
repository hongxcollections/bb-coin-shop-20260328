# 大BB錢幣店 hongxcollections 網站使用指南

## 關於本站
hongxcollections（大BB錢幣店）係香港專業錢幣、紙鈔、郵票網上拍賣 + 出售平台。買家可以競拍或直接購買；商家可以開店上架。

## 兩種上架模式
1. **拍賣（Auctions）** — 設起拍價同結標時間，買家競價，最高價得標。
2. **出售商品（Merchant Products）** — 定價直銷，買家見啱價直接落單。

## 註冊 / 登入
- 點右上角「登入」，可用 email 註冊（Google 登入暫時關閉，現有 Google 用戶仍可繼續登入）。
- 部分功能（出價、收藏、購買、開店）需要先登入。

## 點樣出價（買家）
- 進入拍賣頁面後有 **兩種出價方式**：
  1. **直接出價（手動）**：每口加價必須高於目前最高價 + 最小加價幅度，自己即時出。
  2. **代理出價（自動跟標）**：你輸入「最高願意俾嘅金額」，系統會幫你自動跟標，每次只比對手高一口加幅，直到去到你個上限為止。被超越時系統會通知你。
- 結標時最高出價者得標，系統會自動通知。
- **防狙擊**：結標前 2 分鐘有人出價，會自動延長 2 分鐘。

## 代理出價（自動跟標）會員等級限制
- 🥉 **銅牌**：每月 3 次代理出價額度（用完要等下個月或升級）。
- 🥈 **銀牌**：代理出價次數無限，但**單次上限 HK$5,000**。另外可以用「匿名出價」。
- 🥇 **金牌 / 💎 VIP**：代理出價次數同金額**無上限**，匿名出價預設打開。
- 升級條件可以喺「會員權益 / Member Benefits」頁面睇到。

## 得標 / 付款
- 得標後系統會通知買家及商家。
- 喺「我的得標紀錄（Bid History）」可以查到所有得標。
- 付款方式由商家決定（FPS、銀行過數、面交等），可以喺得標頁直接撳掣 WhatsApp / Facebook Messenger 聯絡商家（會自動 copy 訊息同商品連結）。

## 運費
- 由商家自行決定，請喺出價或落單前 WhatsApp 商家確認。

## 收藏
- 撳商品頁面嘅 ❤️ 即可收藏，喺「我的收藏」可以查閱。

## 會員等級
- 系統會根據出價數、得標數、90 日消費自動升降級：銅 → 銀 → 金 → VIP。
- 長期唔活躍（預設 90 日）會自動降級。
- 銀牌 / 金牌 / VIP 享現金回贈（cashback），詳情入「會員權益」頁。

## 📌 點樣成為商戶（重要！全流程）
成為商戶 **唔止填表咁簡單**，要完成以下幾步：

### 第 1 步：提交商戶申請
1. 登入後喺右上角選單撳「申請成為商戶 / Merchant Apply」。
2. 填寫：商戶名稱、自我介紹、WhatsApp 號碼、商戶頭像（可選）。
3. 提交後等管理員聯絡。

### 第 2 步：等管理員審核 + 聯絡
- 管理員會主動聯絡你（通常 1-3 個工作天）討論：
  - 你想賣咩類型嘅貨
  - 適合你嘅 **訂閱計劃**
  - 需要交嘅 **保證金金額**

### 第 3 步：訂閱月費 / 年費計劃 ⭐（必須）
- **發佈拍賣 / 出售商品 之前一定要先有有效嘅訂閱計劃**，冇訂閱係上唔到架。
- 入「訂閱計劃 / Subscription Plans」頁面揀適合嘅 plan：
  - 不同計劃有不同上架配額、影片配額、置頂位等。
  - 月費 / 年費二選一，年費通常會抵啲。
- 選定 plan → 上載付款證明 → 等管理員確認開通。
- 喺「商戶後台 Dashboard」可以見到目前計劃 + 到期日。

### 第 4 步：繳交保證金（按金）⭐（必須）
- 訂閱開通後仲要 **交保證金**（Deposit）先可以開始上架。
- 金額由管理員根據貨品類型同價值設定。
- 用途：保障買家，如果發生退款 / 糾紛 / 違規會由保證金扣。
- 入「商戶後台 → 保證金」頁面可以充值同查記錄（記錄已分頁顯示，每頁 10 筆）。
- **保證金不足會被暫停上架**，記得補返。

### 第 5 步：開始上架
- 完成以上步驟後，入「商戶拍賣 / 商戶商品」頁撳「新增」即可上架。
- 上架要填：標題、描述、起拍價／售價、結標時間（拍賣）、分類、相片（最多 6 張）、短片（可選）等。

> ⚠️ 如果你問「點上架但又冇訂閱 / 冇交保證金」，畫面會顯示「需要訂閱計劃」或「保證金不足」嘅提示，跟住嗰啲指引去做就得。

## 影片配額
- 商戶每月可上載影片條數同每條最長秒數會根據訂閱計劃決定。
- 喺上架頁面會顯示「本月剩餘 X/Y 條 · 每條最長 Z 秒」。
- 超出配額需等下個月或升級計劃。

## 訂閱續費 / 過期
- 計劃到期後會被標記為「已過期」，唔可以再發佈新拍賣。
- 系統會提前通知，記得續費。
- 過期前已上架嘅商品唔受影響，但唔可以新增。

## 退款
- 買家對已成交訂單有問題，可以喺訂單頁面提交退款申請。
- 商家會收到通知，雙方可以溝通；管理員會仲裁。
- 如判定退款，款項會由商家保證金扣返畀買家。

## Facebook 分享
- 商家可以喺「商家拍賣」或「商家出售商品」頁面，用「批量分享」一次過分享多個商品到 Facebook、WhatsApp 群組。
- 可以喺「商家設定」自訂分享文案模板。
- 亦可用 ✨ AI 文案功能即時生成獨特文案。

## AI 錢幣鑑定（試驗功能）
- 入「Coin Analysis / AI 鑑定」頁面，上載一張清晰錢幣相，AI 會分析背景資料、年份、版本等。
- 鑑定純屬參考，唔代表官方估值。

## 常見問題
- **忘記密碼**：登入頁面撳「忘記密碼」，輸入 email 接收重設連結。
- **點解上架掣 disable / 顯示「需要訂閱計劃」**：請完成（1）通過商戶審核（2）訂閱有效計劃（3）保證金充足。三項缺一不可。
- **點解代理出價設唔到**：可能（1）你係銅牌且本月 3 次配額用晒（2）你係銀牌但金額超過 $5,000 上限（3）你輸入嘅金額低於最低出價。升級會員可解鎖。
- **影片上載失敗**：檢查格式（建議 MP4）同長度（不可超過配額限制）。
- **Facebook 分享預覽資料唔啱**：用 [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/) 嘅「Scrape Again」清快取。
